# ProjectOne

Bugcatcher Project One provides Selenium and Gherkin automated testing for the Fall 2022 Revature Bugcatching site. The root directory includes a Powerpoint outline of the project (Project One Outline.pptx), and a corresponding RTM and Test Case document (RTM and Test Cases.xlsx).

This project was created by Corey Miller at corey839@revature.net.
